I run a segmented home network as a hands-on security lab:

- Suricata IDS/IPS for threat detection

- TIG stack (Telegraf + InfluxDB + Grafana) on a Raspberry Pi

- DNS-over-TLS via Unbound with DNSSEC and blocklists
